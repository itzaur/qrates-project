## ✨ Run it locally

To run the project, follow the steps bellow after cloning the repository:

```bash
# Install dependencies
npm install

# Run in development mode
npm run build-dev

# Run in production mode
npm run build

# Run local server
npm start
```

Then access the application on http://localhost:8080/
